How to Speed Up Windows 11 & Remove Lag (Ultimate Guide)

Your PC has capable hardware but Windows 11 feels sluggish. Apps hesitate, windows stutter when dragged, and games don't hit the FPS your hardware should deliver. Here's how to fix it in about 15 minutes without spending anything.

July 12, 2026 Melalew Mengistu

A client brought in a laptop with a Ryzen 7 processor and 16GB of RAM specs that should feel fast. But opening File Explorer took two seconds. Dragging windows left visible stuttering. Task Manager showed 85% RAM usage at idle with 47 background processes running. After 20 minutes of cleaning up startup programs, disabling transparency effects, and switching the power plan, the same laptop dropped to 52% RAM usage at idle and File Explorer opened instantly.

Windows 11 ships with settings optimized for visual polish, not speed. The gap between "out of the box" and "properly configured" is significant, and most users never bridge it.

Why Windows 11 Feels Slow Out of the Box

Windows 11 uses your GPU to render transparency effects (the blur behind the Start menu and taskbar), window open/close animations, and shadow effects. On integrated graphics or older GPUs, this steals resources from your actual applications.

On top of that, OEMs (Dell, HP, Lenovo, ASUS) preload their machines with vendor software utility suites, update tools, trial antivirus, and app stores that all run background services. A fresh Windows 11 install from a Microsoft ISO uses noticeably less RAM at idle than a branded OEM install with the same hardware.

The fixes below address both layers: the visual overhead from Windows itself and the background clutter from installed software.

What We'll Change
  • Power plan: Switch from "Balanced" to "Best performance" so the CPU doesn't throttle unnecessarily.
  • Startup apps: Remove programs that load at boot and run in the background for no reason.
  • Visual effects: Disable transparency, animations, and shadows that consume GPU resources.
  • Background services: Disable unnecessary Windows features and third-party services.
Who This Guide Is For
  • Anyone noticing input lag, window stuttering, or slow app launches on Windows 11.
  • Gamers who aren't getting the FPS their hardware should deliver.
  • Laptop users dealing with excessive fan noise or battery drain from background processes.
Optimization Layers

We tackle this in three tiers. Start with tier 1 it takes two minutes and produces the most noticeable difference. Move to tier 2 if you want more. Tier 3 is for users comfortable with advanced tools.

1 Power settings and startup cleanup the biggest impact for the least effort.
2 Visual effects and Windows features disabling what you don't see or use.
3 Advanced tweaks registry edits, BIOS settings, and service configuration.

Tier 1

Switch to High Performance Power Plan

Windows 11 defaults to the "Balanced" power plan, which actively throttles your CPU frequency to save power. On a desktop, this saves negligible electricity but introduces micro-stutters when the CPU needs to ramp up from a low frequency. On a laptop, you can switch to "Best performance" when plugged in and "Balanced" on battery.

1

Open Power Settings

Press Win + I to open Settings. Go to System → Power & battery.

2

Set power mode to Best Performance

Find the "Power mode" dropdown and change it from "Balanced" to Best performance. This takes effect immediately no restart needed.

Expected result: The CPU stays at a higher base clock, eliminating the brief lag that occurs when it needs to scale up from a low-power state. Gamers typically see 3 - 8% more consistent frame rates.

On laptops: "Best performance" will drain your battery faster. If you're on battery often, use "Best performance" while plugged in and switch back to "Balanced" when you unplug. Windows doesn't let you set different modes for AC vs battery through the Settings UI, but you can create a custom power plan in Control Panel that does this.

Tier 1

Disable Unnecessary Startup Apps

Many applications register themselves to launch at boot and run continuously in the background. Spotify, Slack, Discord, Adobe updaters, and vendor utility tools all do this. Each one consumes RAM and CPU cycles. On a machine with 8GB of RAM, five unnecessary startup apps can eat 1 - 2GB before you open anything yourself.

1

Open Task Manager

Press Ctrl + Shift + Esc. If Task Manager opens in compact mode, click "More details" at the bottom.

2

Review the Startup tab

Click the Startup tab (the speedometer icon on the left sidebar). You'll see every app configured to launch at boot, along with its "Startup impact" rating. Disable anything you don't need running immediately especially apps rated "High" impact.

3

What to keep and what to disable

Keep enabled: Your antivirus, Windows security components, and hardware drivers (like audio or trackpad utilities if they affect functionality). Disable: Chat apps, game launchers, cloud storage sync, update checkers, and any vendor "optimization" tools. You can still open these apps manually when you need them.

Don't disable your antivirus at startup. If you use Windows Defender (now Microsoft Defender), it's managed by a Windows service and won't appear in the Startup tab so you can't accidentally disable it there. But if you use a third-party antivirus like Bitdefender or Kaspersky, leave its startup entry enabled or you'll have a gap in protection during boot.

Tier 2

Turn Off Visual Effects

Windows 11 renders transparency blur on the Start menu, taskbar, and window title bars using your GPU. It also animates every window open, close, minimize, and maximize. On systems with integrated graphics or older dedicated GPUs, these effects consume 3 - 8% of GPU resources continuously resources your applications or games could use instead.

1

Disable Transparency Effects

Open Settings → Personalization → Colors. Scroll down and toggle off Transparency effects. The taskbar and Start menu will change from translucent to solid immediately.

2

Disable Animation Effects (System Settings)

Go to Settings → Accessibility → Visual effects. Toggle off Animation effects. This removes the slide-in and fade-out animations for windows and menus. Windows still functions identically it just appears and disappears instead of animating.

3

Adjust Performance Options (Advanced)

Press Win + R, type sysdm.cpl, and press Enter. Click the Advanced tab, then click Settings under Performance. Select Adjust for best performance to disable all visual effects, or manually uncheck specific ones like "Animate windows when minimizing and maximizing" and "Show shadows under windows" if you want to keep some visual polish.

Expected result: Window dragging becomes perfectly smooth, apps feel snappier to open, and gamers on integrated graphics typically gain 5 - 15 FPS depending on the game.

Compromise approach: If "Adjust for best performance" makes Windows look too flat, enable just "Smooth edges of screen fonts" and "Use drop shadows for icon labels on the desktop." These two have near-zero performance cost but make text and icons noticeably easier to read.

Tier 2

Disable Unnecessary Windows Features

Windows 11 enables several optional features by default that most users never touch. These features run background services or reserve system resources even when inactive. Disabling the ones you don't use reduces background overhead.

1

Open Windows Features

Press Win + R, type optionalfeatures, and press Enter. This opens the "Turn Windows features on or off" dialog. It may take a few seconds to load.

2

Disable what you don't need

Safe to disable for most users: Internet Explorer 11 (legacy, unused), Windows PowerShell 2.0 (outdated version, PowerShell 5+ is already installed), Print and Document Services (if you don't share printers over a network), Windows Sandbox (unless you actively test software in sandboxes), and Hyper-V (unless you run virtual machines this one reserves RAM at boot).

3

Restart when prompted

Click OK. Windows will apply the changes and prompt you to restart. The changes take effect after reboot.

Don't disable: .NET Framework (many applications depend on it), Windows Media Player (some apps use its components), or Remote Desktop Services (if you ever remote into this machine). When in doubt, leave it enabled the performance gain from any single feature is small.

Tier 3

Advanced: Registry Edits and BIOS Settings

These tweaks target deeper system behavior mouse input processing, CPU power management at the hardware level, and Windows animation timing. They produce measurable differences but require more care since you're editing system-level settings.

Disable Mouse Pointer Acceleration (for gamers)

Windows applies pointer acceleration by default, which changes the distance your cursor moves based on how fast you move the mouse. Gamers almost universally disable this because it makes aim inconsistent. Go to Settings → Bluetooth & devices → Mouse → click Additional mouse settings → Pointer Options tab → uncheck Enhance pointer precision.

Enable XMP/EXPO in BIOS (for RAM speed)

Your RAM may be running below its rated speed. DDR5 RAM rated for 6000MHz often runs at 4800MHz by default because BIOS doesn't enable the XMP (Intel) or EXPO (AMD) profile automatically. Reboot, enter BIOS (usually by pressing Delete or F2 during boot), find the XMP/EXPO option under memory settings, and enable it. This can improve gaming performance by 5–10% depending on the game and how memory-bound it is.

Disable Game DVR (if you don't use it)

Windows 11's Game DVR feature (accessed via Win + G) runs a background recording service even when you're not actively recording. If you use OBS or a dedicated capture card instead, disable it. Open Settings → Gaming → Captures → toggle off Record in the background while I'm playing a game.

BIOS changes carry real risk. Incorrect BIOS settings can prevent your system from booting. Only change settings you understand. If something goes wrong, BIOS almost always has a "Reset to defaults" option you can access from the same menu. When enabling XMP/EXPO, if your system becomes unstable, disable it and run your RAM at the default speed stability matters more than a few extra FPS.

Registry edits: Many "speed up Windows 11" guides recommend registry changes like disabling Nagle's algorithm or modifying the TCP autotuning level. These primarily affect network latency, not general system speed, and can cause issues on some networks. We've omitted them here because the risk-to-reward ratio is poor for most users. If you have a specific network latency problem, that calls for a targeted guide, not blanket registry changes.
